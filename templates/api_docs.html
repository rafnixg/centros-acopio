📡 Endpoints Disponibles

Todos los endpoints devuelven JSON. No requieren autenticación a menos que se indique.

GET /api/centros

Listar todos los centros de acopio. Acepta filtros opcionales vía query string:

  • ?q= — Búsqueda por nombre, ciudad o dirección
  • ?estado= — Filtrar por estado (ej: Distrito Capital)
  • ?producto= — Filtrar por tipo de producto
  • ?estado_centro= — Filtrar por Activo/Lleno/Pausado/Cerrado
curl https://centrosacopio.rafnixg.dev/api/centros?estado=Carabobo
GET /api/centros/{id}

Obtener detalle de un centro específico.

curl https://centrosacopio.rafnixg.dev/api/centros/1
POST /api/centros

Registrar un nuevo centro de acopio.

curl -X POST https://centrosacopio.rafnixg.dev/api/centros \
  -H "Content-Type: application/json" \
  -d '{"nombre":"Mi Centro","estado":"Carabobo","ciudad":"Valencia","direccion":"Av. Principal","telefono":"0412-0000000","responsable":"Yo"}'
GET /api/estadisticas

Obtener estadísticas: total de centros, por estado, por producto, centros recientes.

GET /api/estados

Lista de 24 estados de Venezuela.
